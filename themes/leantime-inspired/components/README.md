# Components

This folder holds framework-neutral component styling and guidance for the
`leantime-inspired` theme.

Current starter:

- `base.css`: page, shell, sidebar, card, button, form control, table, badge,
  alert, focus, and responsive shell primitives.

Initial component groups to expand:

- buttons;
- forms;
- tables;
- cards;
- navigation;
- tabs;
- badges;
- feedback;
- modals;
- loading and empty states;
- accessibility helpers.

Component styles should consume tokens from `../tokens/`.
