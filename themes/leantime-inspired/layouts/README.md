# Layouts

This folder holds reusable surface shells for the `leantime-inspired`
theme.

Current starter:

- `document.css`: single-page design docs, architecture docs, briefs, and
  operations plans.

Initial layout targets to expand:

- `app-shell.html`: authenticated internal tools and product UIs.
- `api-platform.html`: API documentation, endpoint reference, auth, and
  playground surfaces.
- `dashboard.html`: operational dashboards, metric views, and admin surfaces.

Layouts should be framework-neutral examples unless a product-specific
technology profile requires a framework implementation.
