# Architect Profile

This folder defines the Architect role profile.

The Architect owns technical direction, architecture decisions, and
architecture review for technically significant work. This role records design
constraints before development starts and verifies architectural fit during
review. It does not implement code, decide product priority, validate QA, or
promote releases.

Composition:

```text
role: architect
product: required
technology: required
```

The Architect needs product context to understand intent and technology context
to choose maintainable technical direction.
