# Technical Lead Profile

This folder defines the Technical Lead role profile.

The Technical Lead coordinates the development lifecycle after product, UX, and
architecture inputs are accepted. This role owns development planning, GitHub
Issue hygiene, defect backlog triage, development/test readiness within its
authority, TDD evidence enforcement, and process enforcement across Developer,
QA, Release, and Operations handoffs.

Composition:

```text
role: tech-lead
product: required
technology: required for environment, validation, and issue planning rules
```

The Technical Lead needs product context to preserve intent and technology
context to plan executable work. It does not define product scope, make
architecture decisions, write application code, run acceptance tests, assemble
release branches, or promote releases.
